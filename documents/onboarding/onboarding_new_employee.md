# Onboarding Checklist (Week 1)

Day 1:
- Access to Git + Figma + Notion
- Intro to codebase structure
- Security briefing (no secrets in repo)

Day 2–3:
- First task: small UI fix
- Code review + style rules
- Daily short sync

Day 4–5:
- Deliver first feature
- Write short documentation
- Demo to team
